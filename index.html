Welcome!

I have had a hand in making many things ranging from a User Generated Content Focused Virtual Realty application to a Text Based Massively Multiplayer Role Playing Game. Below you will find a showcase of some of my favorite projects that I've worked on. I hope you enjoy them too!

My Github Current Project

Discord Duels is a text based MMORPG (Massively Multiplayer Online Role Playing Game) that is playable through the Discord application's Mobile and Desktop interfaces.

Locus is location based image sharing service that aims at allowing people to leave behind personal memories as they travel through the world.

Xeno is a mobile application developed with the intent of letting people step into the shoes of others through personal virtual reality experiences.

Balloon is a web based application that allows users to transfer files quickly from device to device.

Trends is a web service that mixes information gathered from the Google Trends API and the Twitter API to provide a small summary of what people are talking about in the world. This service is currently down as server space is being used primarily for Discord Duels.

Challonge Bot is a Discord bot user that allows users to create and run tournaments from within the Discord application's Mobile and Desktop interfaces using a custom made wrapper for the Challonge Tournament API. This bot is currently down as server space is being used primarily for Discord Duels.

Absolver bot is Discord bot user that allows users to access the official wikia pages for the game Absolver, by Sloclap, from within the Discord application's Mobile and Desktop interfaces. It is currently on the offical Discord server for Absolver which is managed by the Sloclap team themselves. This bot is currently down as server space is being used primarily for Discord Duels.

Ace Ninjas is platformer rogue-like with a simplistic pixelated art style that that focuses on exploration of randomy generated terrain and ability based combat. This game is currently still in development.

Bullet Hell, the first game I ever created, is a endless space shooter in which the player must defend their space station from alien siege using different ships and abilities to over come obstacles they face.
